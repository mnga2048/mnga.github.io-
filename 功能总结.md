# 功能总结

## 一、页面布局与导航

### 侧边栏导航树
- 固定在页面左侧，显示完整的文件目录结构
- 支持**多级文件夹展开/折叠**，点击箭头或文件夹名切换
- 当前浏览页面自动**高亮显示**
- 折叠状态通过 `localStorage` 持久化保存

### 面包屑导航
- 子目录页和 Markdown 查看页顶部显示当前路径
- 每一级路径可点击跳转

### 移动端适配
- 768px 以下自动切换为移动端布局
- 侧边栏变为滑出式抽屉

---

## 二、搜索功能

### 导览搜索（文件名搜索）
- 侧边栏顶部搜索框，输入关键词实时过滤文件名
- 空搜索时恢复全部显示，文件夹自动隐藏/显示

### 全局搜索
- 快捷键 `Ctrl+Shift+K` 打开搜索面板
- 搜索所有笔记的文件名，显示匹配结果列表
- 支持上下方向键选择、Enter 打开、Esc 关闭

### 文件内容搜索
- 笔记页面顶部搜索栏，支持 `Ctrl+F` 快捷键
- 匹配文本高亮显示，上下/左右方向键跳转
- Enter 跳转下一个匹配，Esc 清空搜索

---

## 三、主题系统

### 亮色/暗色主题切换
- 默认纸张色护眼主题，一键切换暗色主题
- 快捷键 `T` 切换
- 主题偏好 localStorage 持久化保存

---

## 四、Markdown 查看器

### 动态 MD 渲染
- `md.html?file=<路径>` 加载并渲染任意 `.md` 文件
- 使用 marked.js 解析，支持 GFM 扩展（表格、任务列表等）

### KaTeX 数学公式
- 支持 `$行内公式$` 和 `$$块级公式$$`
- CDN 引入 KaTeX 自动渲染

---

## 五、阅读辅助

### 阅读进度条
- 页面顶部金色进度条，随滚动实时更新

### 文章目录（ToC）
- 侧边栏自动提取 h1/h2/h3 生成可跳转目录
- 滚动时当前章节自动高亮

### 字数与阅读时间
- 面包屑下方显示 "xxx 字 · 约 x 分钟"

### 阅读进度记忆
- 记住每篇笔记上次阅读的滚动位置
- 下次打开自动恢复

### 返回顶部
- 滚动超过一屏后右下角出现按钮
- 点击平滑回到页面顶部

---

## 六、文件管理

### 下载功能
- 右上角 MD 按钮：直接下载 Markdown 原文件
- 右上角 PDF 按钮：调起浏览器打印（可保存为 PDF）

### 收藏笔记
- 右上角 ☆ 按钮收藏当前笔记
- 侧边栏显示收藏列表，点击快速跳转

### 最近浏览
- 侧边栏记录最近 15 篇浏览过的笔记
- 自动去重，按时间倒序排列

---

## 七、自动页面生成

### generate.py
- 扫描目录结构，自动生成首页和所有子目录的 index.html
- 为每个目录生成文件列表和子目录卡片
- 根据文件扩展名自动检测技术标签

### config.json 配置
| 字段 | 说明 |
|------|------|
| `name` | 用户名 |
| `bio` | 个人简介 |
| `avatar_url` | 头像 URL |
| `social` | GitHub、Email 等社交链接 |
| `skills` | 技能标签列表 |
| `exclude_dirs` | 排除的目录名列表 |

---

## 八、快捷键一览

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+K` | 聚焦侧边栏文件名搜索 |
| `Ctrl+Shift+K` | 打开全局搜索面板 |
| `Ctrl+F` | 文件内容搜索 |
| `T` | 切换主题 |
| `?` | 显示快捷键帮助面板 |
| `Esc` | 关闭弹窗/清空搜索 |
| `Enter` | 搜索结果中打开文件 / 跳转下一个匹配 |
| `←` `→` `↑` `↓` | 在搜索结果中切换 |

---

## 九、其他

### 打印优化
- 打印时自动隐藏侧边栏、搜索栏、进度条等 UI 元素
- 仅保留纯净内容，适配黑白打印

### 键盘快捷键帮助
- 按 `?` 弹出快捷键说明面板

### 目录排除
- 当前仅排除 `.obsidian` 和 `附件` 文件夹
- 可通过 config.json 的 `exclude_dirs` 字段自定义
