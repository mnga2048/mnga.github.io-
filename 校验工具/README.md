# 校验工具

一个轻量级的通信协议校验计算工具，纯 HTML 实现，浏览器直接打开即用，无需安装任何依赖。

## 功能

### 1. Modbus RTU CRC16

计算/校验标准 Modbus RTU CRC16。

- **多项式**: 0x8005（反转 0xA001）
- **初始值**: 0xFFFF
- **输入/输出反转**: 是
- **结果异或**: 0x0000

支持：
- 计算 CRC：输入数据字节，输出 CRC16 结果、高低字节、完整 Modbus 帧
- 校验 CRC：输入含 CRC 的完整帧，验证最后两字节是否正确
- 历史记录、一键复制、Ctrl+Enter 快捷计算

### 2. 校验和（取反）

适用于 `FF FF` 固定帧头的串行总线协议（如飞特舵机协议）。

- **帧格式**: `FF FF ID LEN CMD [Params] CheckSum`
- **校验算法**: `CheckSum = ~(ID + LEN + CMD + Params...) & 0xFF`

支持：
- 计算校验和：输入 FF FF 后的数据部分，输出校验字节和完整帧
- 校验整帧：输入完整帧（可含/不含 FF FF 帧头），验证校验字节
- 历史记录、一键复制

### 3. 主题切换

- 浅色 / 深色主题一键切换
- 默认浅色主题
- 主题偏好保存在浏览器 localStorage，刷新不丢失

### 输入格式

支持多种十六进制输入格式：
- 空格分隔: `01 03 00 00 00 0A`
- 连续输入: `010300000A`
- 带 0x 前缀: `0x01 0x03 0x00 0x00 0x00 0x0A`
- 混合格式均可识别
