## 1. 核心区别

### 快速对比表

| 特性 | **D项（微分）** | **输入滤波** |
|------|---------------|-------------|
| **作用对象** | 误差的**变化率** | 测量信号的**噪声** |
| **频域特性** | **高通滤波器** ⬆ | **低通滤波器** ⬇ |
| **相位特性** | **超前**（预测） | **滞后**（延迟） |
| **对噪声影响** | **放大** ❌ | **抑制** ✅ |
| **主要目的** | 减少超调，加快收敛 | 提高测量精度 |
| **副作用** | 引入高频噪声 | 引入相位延迟 |
| **适用场景** | 低速、惯性大、测量准 | 高速、噪声大、测量糙 |

### 本质区别

```
D项：预测未来，主动干预
      "误差在减小，我要提前刹车"

滤波：回顾过去，平滑信号
      "信号在抖动，我要平均处理"
```

## 2. D项（微分）详解

### 2.1 数学表达

#### 连续形式
```
D输出 = Kd × d(误差)/dt
```

#### 离散形式（本项目实现）
```c
// PID.c:163
PID->KdOutput = (PID->Kd * (PID->Ek1 - PID->Ek)) >> PID->Shift;

// 展开
D输出 = Kd × (上一次误差 - 当前误差)
```

### 2.2 频域特性

#### 传递函数
```
G(s) = Kd × s
```

#### 幅频特性
```
|G(jω)| = Kd × ω

频率 ω 越高 → 幅值越大
```

#### 波特图示意
```
增益
  ↑
  │        ╱
  │       ╱
  │      ╱  ← 高频放大（+20dB/dec）
  │     ╱
  │    ╱
  └───┴──────→ 频率 (log)
     ω=0
```

**结论**：D项本质是**高通滤波器**，会放大高频信号

### 2.3 时域行为

#### 工作原理
```
场景：转速阶跃响应 0 → 5000 RPM

t=0ms:   误差=5000, D=0         ← 初始状态
t=10ms:  误差=4000, D=+1000Kd   ← 误差快速减小，D输出正，加速收敛
t=20ms:  误差=3000, D=+1000Kd   ← 继续加速
t=30ms:  误差=2000, D=+1000Kd
t=40ms:  误差=1500, D=+500Kd    ← 误差减小变慢，D减小
t=50ms:  误差=1000, D=+500Kd
t=60ms:  误差=800,  D=+200Kd
t=70ms:  误差=900,  D=-100Kd    ← 误差反弹，D输出负，阻止上升
t=80ms:  误差=850,  D=+50Kd     ← 误差再次减小
```

#### 作用效果

**优点**：
- ✅ 预测误差趋势，提前干预
- ✅ 减少超调
- ✅ 加快收敛速度
- ✅ 抑制低频震荡

**缺点**：
- ❌ 放大测量噪声
- ❌ 对高频敏感
- ❌ 容易引入抖动

## 3. 输入滤波详解

### 3.1 数学表达

#### 一阶低通滤波器（本项目使用）
```c
// MotorDriver.c:596
*CaptureCnt = FirstOrder_Filter(TimerCapCnt, *CaptureCnt, 40);

// 算法展开
滤波输出 = α × 新值 + (1-α) × 旧值

其中 α = 1 / 滤波系数
```

#### 差分方程
```
y[n] = α × x[n] + (1-α) × y[n-1]

例如滤波系数=40：
y[n] = (1/40) × x[n] + (39/40) × y[n-1]
     = 0.025 × 新值 + 0.975 × 旧值
```

### 3.2 频域特性

#### 传递函数
```
G(s) = 1 / (τs + 1)

其中 τ = 时间常数
```

#### 幅频特性
```
|G(jω)| = 1 / √(1 + (ωτ)²)

频率 ω 越高 → 幅值越小
```

#### 波特图示意
```
增益
  ↑
  │
1 │------╲
  │        ╲
  │         ╲  ← 高频衰减（-20dB/dec）
  │          ╲
  │           ╲
  └───┬───────╴──→ 频率 (log)
      ωc
     截止频率
```

**结论**：滤波器是**低通滤波器**，抑制高频信号

### 3.3 时域行为

#### 工作原理
```
场景：霍尔传感器信号抖动

真实转速：5000 RPM

无滤波的测量值：
t=0ms:  5000
t=10ms: 5050  ← 抖动+50
t=20ms: 4950  ← 抖动-50
t=30ms: 5030  ← 抖动+30
t=40ms: 4970  ← 抖动-30

滤波后的值（系数=40）：
t=0ms:  5000
t=10ms: 5001  ← 5000×0.975 + 5050×0.025
t=20ms: 5000  ← 5001×0.975 + 4950×0.025
t=30ms: 5000  ← 5000×0.975 + 5030×0.025
t=40ms: 5000  ← 5000×0.975 + 4970×0.025
```

#### 作用效果

**优点**：
- ✅ 抑制测量噪声
- ✅ 提高信号质量
- ✅ 减少系统抖动
- ✅ 保护执行器

**缺点**：
- ❌ 引入相位延迟
- ❌ 降低响应速度
- ❌ 可能掩盖真实变化

## 4. 为什么D项会放大噪声？

### 4.1 微分放大噪声的原理

#### 数学推导
```
设真实信号：x(t)
测量噪声：n(t)
测量值：y(t) = x(t) + n(t)

微分运算：
dy/dt = dx/dt + dn/dt

如果噪声是高频振荡：
n(t) = A×sin(ωt)
dn/dt = A×ω×cos(ωt)  ← ω越大，幅值越大！
```

**结论**：噪声频率越高，微分后幅值越大

### 4.2 实际计算示例

#### 场景1：无噪声的理想信号
```c
// 转速平稳上升
t0: 5000 RPM
t1: 5002 RPM  ← 真实变化+2
t2: 5004 RPM  ← 真实变化+2

// 微分计算（假设Kd=1）
D(t1) = 5002 - 5000 = +2
D(t2) = 5004 - 5002 = +2
结果：平滑输出 ✅
```

#### 场景2：有噪声的测量信号
```c
// 真实转速平稳：5000 RPM
// 霍尔传感器抖动：±5 RPM

t0: 5000 RPM
t1: 5005 RPM  ← 抖动+5
t2: 4995 RPM  ← 抖动-5
t3: 5003 RPM  ← 抖动+3

// 微分计算
D(t1) = 5005 - 5000 = +5   ← 5倍噪声！
D(t2) = 4995 - 5005 = -10  ← 10倍噪声！！
D(t3) = 5003 - 4995 = +8   ← 8倍噪声！

结果：严重抖动 ❌
```

### 4.3 频域分析

#### D项对噪声的增益
```
|D(jω)| = Kd × ω

噪声频率  增益   噪声幅值
1 Hz     Kd×1   1×Kd
10 Hz    Kd×10  10×Kd   ← 放大10倍
100 Hz   Kd×100 100×Kd  ← 放大100倍
1000 Hz  Kd×1000 1000×Kd ← 放大1000倍
```

**结论**：高频噪声会被D项大幅放大

## 5. 滤波对D项的影响

### 5.1 频域分析

#### 系统开环传递函数
```
L(s) = 滤波器 × PID控制器 × 被控对象

无D项：
L(s) = [1/(τs+1)] × (Kp + Ki/s) × G(s)

有D项：
L(s) = [1/(τs+1)] × (Kp + Ki/s + Kd×s) × G(s)
     = [1/(τs+1)] × Kp × G(s)                    ← P项
     + [1/(τs+1)] × Ki/s × G(s)                  ← I项
     + [1/(τs+1)] × Kd×s × G(s)                  ← D项
     = Kd × [s/(τs+1)] × G(s)
            ↑
       带通滤波器（不再是纯高通）
```

#### D项的等效传递函数
```
无滤波：D项 = Kd×s             （纯高通）
有滤波：D项 = Kd×s/(τs+1)     （带通）

幅频特性对比：

增益
  ↑
  │   无滤波  ┌───  ← 高频无限放大
  │          ╱
  │         ╱
  │   有滤波╱───    ← 高频被限制
  │       ╱
  └───┬──╱─────→ 频率
      1/τ
     截止频率
```

**结论**：滤波器将D项的"纯高通"变成"带通"，限制高频放大

### 5.2 项目中的实际配置

#### 低速段（使用D项）
```c
// PID.c:596
*CaptureCnt = FirstOrder_Filter(TimerCapCnt, *CaptureCnt, 40);

滤波系数 = 40
截止频率 fc ≈ 采样频率 / (2π×40)
          ≈ 1000 Hz / 251
          ≈ 4 Hz

效果：4 Hz以下信号通过，4 Hz以上噪声被抑制
```

#### 高速段（不用D项）
```c
// PID.c:617
*CaptureCnt = FirstOrder_Filter(TimerCapCnt, *CaptureCnt, 128);

滤波系数 = 128
截止频率 fc ≈ 1000 Hz / (2π×128)
          ≈ 1.2 Hz

效果：更强滤波，因为高速时噪声更大
```

## 8. 设计原则

### 8.1 何时使用D项？

#### ✅ 推荐使用D项的场景

| 场景 | 原因 | 推荐Kd值 |
|-----|------|---------|
| **低速启动（<2000 RPM）** | 惯性影响大，D可抑制超调 | 0.2~0.4 |
| **大惯性负载** | 需要D抑制震荡 | 0.3~0.5 |
| **编码器反馈** | 测量精度高，噪声小 | 0.1~0.3 |
| **快速响应需求** | D可加快收敛 | 0.2~0.4 |
| **低频震荡** | D可抑制低频振荡 | 0.3~0.6 |

#### ❌ 不推荐使用D项的场景

| 场景 | 原因 | 替代方案 |
|-----|------|---------|
| **高速运行（>6000 RPM）** | 测量噪声大，D会放大 | 使用纯PI+强滤波 |
| **霍尔传感器反馈** | 测量粗糙，噪声大 | 加强滤波，不用D |
| **高频噪声环境** | D会严重放大噪声 | 使用低通滤波 |
| **稳态精度优先** | D会引入抖动 | 使用纯PI |
| **计算资源受限** | 增加计算量 | 简化为PI |

### 8.2 滤波强度选择

#### 滤波系数对照表

| 滤波系数 | 截止频率 | 延迟 | 适用场景 |
|---------|---------|------|---------|
| 16 | ~10 Hz | 小 | 低速、有D项、编码器 |
| 40 | ~4 Hz | 中 | **低速段、有D项（推荐）** |
| 64 | ~2.5 Hz | 中大 | 中速、无D项 |
| 128 | ~1.2 Hz | 大 | **高速段、无D项（推荐）** |
| 256 | ~0.6 Hz | 很大 | 极高噪声、牺牲响应 |

#### 选择原则

```
1. D项存在时 → 滤波强一些（补偿噪声放大）
   推荐：40-64

2. D项不存在时 → 滤波弱一些（减少延迟）
   推荐：64-128

3. 低速段 → 滤波弱一些（保持响应速度）
   推荐：40-64

4. 高速段 → 滤波强一些（抑制高频噪声）
   推荐：128-256

5. 测量粗糙（霍尔） → 滤波强一些
   推荐：128-256

6. 测量精确（编码器） → 滤波弱一些
   推荐：16-40
```

### 8.3 调试流程

#### 步骤1：确定D项需求
```
问题诊断：
  - 超调>10%    → 需要D项
  - 震荡>2次    → 需要D项
  - 响应慢      → 可能需要D项
  - 高速运行    → 不需要D项
  - 噪声大      → 不需要D项
```

#### 步骤2：调整滤波强度
```
有D项时：
  - 观察PWM输出抖动
  - 抖动大 → 增大滤波系数
  - 抖动小但响应慢 → 减小滤波系数
  - 目标：PWM输出变化<5%

无D项时：
  - 观察转速测量值
  - 噪声大 → 增大滤波系数
  - 响应慢 → 减小滤波系数
  - 目标：转速测量抖动<1%
```

#### 步骤3：验证配合效果
```
测试项目：
  ✓ 阶跃响应：超调<5%，无震荡
  ✓ 稳态运行：转速波动<1%
  ✓ 负载突变：恢复快，无超调
  ✓ PWM输出：无明显抖动
```

## 10. 总结

### 核心要点

1. **D项和滤波是互补关系，不是替代关系**
   - D项：主动预测，减少超调
   - 滤波：被动平滑，抑制噪声

2. **D项会放大噪声，必须配合滤波使用**
   - 高频噪声 → D项放大10-100倍
   - 滤波器 → 抑制高频噪声
   - 配合使用 → 平衡性能与稳定性

3. **低速用D，高速不用**
   - 低速：惯性大，需要D抑制震荡
   - 高速：噪声大，避免D放大噪声

4. **滤波强度根据D项调整**
   - 有D项 → 滤波强（系数40-64）
   - 无D项 → 滤波弱（系数64-128）

### 决策树

```
是否使用D项？
  │
  ├─ 测量噪声大？
  │    ├─ 是 → ❌ 不用D
  │    └─ 否 → 继续
  │
  ├─ 转速>6000 RPM？
  │    ├─ 是 → ❌ 不用D
  │    └─ 否 → 继续
  │
  ├─ 有超调/震荡？
  │    ├─ 是 → ✅ 使用D（0.2-0.4）
  │    └─ 否 → ❌ 不用D
  │
选择滤波强度？
  │
  ├─ 有D项？
  │    └─ 是 → 滤波系数40-64
  │
  ├─ 高速（>6000）？
  │    └─ 是 → 滤波系数128-256
  │
  └─ 其他 → 滤波系数64-128
```

### 最佳实践

✅ **低速启动**：PID(Kd=0.4) + 滤波(40)
✅ **中速运行**：PID(Kd=0.4) + 滤波(40)
✅ **高速运行**：PI(Kd=0) + 滤波(128)
✅ **编码器反馈**：可用D项，滤波可弱
✅ **霍尔反馈**：慎用D项，滤波要强
