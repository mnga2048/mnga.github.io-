
## 2. 输入滤波详解

### 2.1 定义与位置

**输入滤波**（Input Filtering），也称**反馈滤波**（Feedback Filtering）

```
位置：PID控制器的输入端（反馈通道）

对象：被控量的测量值
      - 转速测量
      - 电流测量
      - 位置测量
      - 压力/温度等
```

### 2.2 数学表达

#### 一阶低通滤波器
```c
// 离散形式
y[n] = α × x[n] + (1-α) × y[n-1]

其中：
  x[n] = 当前测量值
  y[n] = 当前滤波输出
  α    = 滤波系数 (0 < α < 1)

// 项目实现（User_Application.c:14）
int32_t FirstOrder_Filter(int32_t New_Data, int32_t Old_Data, int32_t k)
{
    // k值越大，滤波越强
    // α = 1/k
    return (New_Data + (k-1) * Old_Data) / k;
}
```

#### 连续形式
```
传递函数：G(s) = 1 / (τs + 1)

其中 τ = 时间常数
```

### 2.3 频域特性

#### 幅频特性
```
|G(jω)| = 1 / √(1 + (ωτ)²)

特性：
  - 低频信号（ωτ << 1）：增益 ≈ 1（无衰减）
  - 高频信号（ωτ >> 1）：增益 ≈ 1/(ωτ)（大幅衰减）
  - 截止频率：ωc = 1/τ
```

#### 波特图
```
增益(dB)
  0 ──────╮
           │
           ╲────────
            ╲
             ╲──────
              ╲
  -20 ────────╲─────→ 频率(log)
               ωc
```

### 2.4 时域行为

#### 对阶跃输入的响应
```
输入：单位阶跃（0 → 1）

输出：y(t) = 1 - e^(-t/τ)

时间点    输入      输出      误差
t=0τ      1.0      0.000     100%
t=1τ      1.0      0.632     36.8%
t=2τ      1.0      0.865     13.5%
t=3τ      1.0      0.950     5.0%
t=5τ      1.0      0.993     0.7%
```

#### 对噪声的抑制
```
场景：霍尔传感器抖动

真实转速：5000 RPM
测量噪声：±50 RPM（10 Hz高频）

无滤波：
  测量值：5050 → 4950 → 5030 → 4970 → 5020
  PID误差剧烈波动 → PWM输出抖动 ❌

有滤波（k=40）：
  滤波值：5001 → 5000 → 5000 → 5000 → 5000
  PID误差平滑 → PWM输出稳定 ✅
```

### 2.5 主要作用

#### ✅ 优点

| 作用 | 说明 | 应用场景 |
|-----|------|---------|
| **抑制测量噪声** | 去除传感器高频抖动 | 霍尔传感器、ADC采样 |
| **提高信号质量** | 平滑测量数据 | 转速、电流测量 |
| **保护执行器** | 避免PWM剧烈抖动 | 电机、阀门控制 |
| **减少积分饱和** | 噪声不进入积分项 | 长时间稳定运行 |
| **改善系统稳定性** | 降低高频增益 | 易震荡系统 |

#### ❌ 缺点

| 缺点 | 说明 | 影响 |
|-----|------|------|
| **引入相位延迟** | 输出滞后于输入 | 响应变慢 |
| **降低响应速度** | 时间常数越大越慢 | 动态性能下降 |
| **掩盖真实变化** | 强滤波会滤除有用信号 | 快速扰动响应差 |
| **降低稳定裕度** | 相位滞后可能失稳 | 需要重新调参 |

### 2.6 相位延迟分析

#### 相位频率特性
```
φ(ω) = -arctan(ωτ)

频率   相位延迟   时间延迟
0.1ωc  -5.7°     τ/10
ωc     -45°      τ/1.57
10ωc   -84°      τ/6

举例（τ=100ms）：
  1 Hz信号：延迟 16ms
  10 Hz信号：延迟 159ms
  100 Hz信号：延迟 160ms（接近最大）
```

## 3. 输出滤波详解

### 3.1 定义与位置

**输出滤波**（Output Filtering），也称**前向通道滤波**（Forward Path Filtering）

```
位置：PID控制器的输出端（前向通道）

对象：控制信号
      - PWM占空比
      - 阀门开度
      - 执行器驱动信号
```

### 3.2 数学表达

#### 一阶低通滤波器
```c
// 算法形式（与输入滤波相同）
y[n] = α × x[n] + (1-α) × y[n-1]

但对象不同：
  x[n] = PID计算出的控制量
  y[n] = 实际送给执行器的控制量
```

#### 典型实现
```c
// 伪代码示例
int32_t PID_Output_Filter(int32_t PID_Output, int32_t Last_Output)
{
    // 滤波系数根据执行器特性选择
    int32_t k = 20;  // 中等强度

    return (PID_Output + (k-1) * Last_Output) / k;
}
```

### 3.3 频域特性

#### 传递函数
```
完整开环传递函数：
L(s) = [输出滤波器] × [PID控制器] × [对象]

L(s) = [1/(τo·s + 1)] × [Kp + Ki/s + Kd·s] × G(s)
                    ↑
                   输出滤波
```

#### 幅频特性
```
|L(jω)| = |PID(jω)| × |G(jω)| / √(1 + (ωτo)²)

特性：
  - 低频：PID和对象增益主导
  - 高频：输出滤波器衰减
  - 作用：限制高频控制信号
```

### 3.4 时域行为

#### 对PID输出的平滑
```
场景：PID输出跳变

无输出滤波：
  t0:  PID输出 100 → PWM 100%
  t1:  PID输出 50  → PWM 50%
  执行器突变 → 机械冲击 ❌

有输出滤波（k=20）：
  t0:  PID输出 100 → 滤波后 100 → PWM 100%
  t1:  PID输出 50  → 滤波后 95  → PWM 95%
  t2:              → 滤波后 90  → PWM 90%
  ...
  t10:             → 滤波后 50  → PWM 50%
  执行器平滑 → 无机械冲击 ✅
```

### 3.5 主要作用

#### ✅ 优点

| 作用 | 说明 | 应用场景 |
|-----|------|---------|
| **平滑执行器动作** | 避免突变，减少机械冲击 | 电机启动、换相 |
| **抑制执行器抖动** | 防止高频振荡 | PWM控制、阀门 |
| **减少机械磨损** | 延长执行器寿命 | 频繁动作系统 |
| **避免饱和触发** | 平滑过渡，不超限 | 限幅保护系统 |
| **改善动态响应** | 抑制超调，减少震荡 | 大惯性系统 |

#### ❌ 缺点

| 缺点 | 说明 | 影响 |
|-----|------|------|
| **降低响应速度** | 执行器动作滞后 | 快速响应困难 |
| **引入额外延迟** | 增加系统时间常数 | 相位裕度下降 |
| **可能降低稳定性** | 相位滞后积累 | 需要重新调参 |
| **掩盖控制效果** | 强滤波削弱PID作用 | 动态性能下降 |

### 3.6 与执行器的匹配

#### 不同执行器的滤波策略

| 执行器类型 | 响应速度 | 推荐滤波强度 | 原因 |
|-----------|---------|-------------|------|
| **直流电机** | 中等 | 中等（k=20-40） | 平滑启动，避免电流冲击 |
| **步进电机** | 快 | 弱（k=10-20） | 保持快速响应 |
| **伺服电机** | 快 | 弱或不滤波 | 内环已滤波 |
| **阀门** | 慢 | 强（k=50-100） | 防止水锤效应 |
| **加热器** | 很慢 | 很强（k=100+） | 热惯性大 |

## 4. 核心区别对比

### 4.1 详细对比表

| 对比维度 | **输入滤波** | **输出滤波** |
|---------|------------|-------------|
| **位置** | 反馈通道（输入端） | 前向通道（输出端） |
| **处理对象** | 测量信号（转速、电流） | 控制信号（PWM、阀门） |
| **主要目的** | 抑制测量噪声 | 平滑执行器动作 |
| **作用原理** | 低频通过，高频衰减 | 低频通过，高频衰减 |
| **对噪声** | 抑制 ✅ | 不影响 |
| **对执行器** | 无直接影响 | 保护 ✅ |
| **相位延迟** | 引入延迟 | 引入延迟 |
| **响应速度** | 降低 | 降低 |
| **稳定性** | 通常提高稳定性 | 可能降低稳定性 |
| **适用场景** | 测量噪声大 | 执行器抖动、机械冲击 |
| **设计难度** | 容易（独立设计） | 较难（需考虑对象） |

### 4.2 频域影响对比

#### 输入滤波对开环传递函数的影响
```
L(s) = [PID] × [对象] / (τi·s + 1)
             ↑
           输入滤波

影响：
  - 闭环极点：略微左移（更稳定）
  - 相位裕度：略微下降
  - 剪切频率：基本不变
```

#### 输出滤波对开环传递函数的影响
```
L(s) = [1/(τo·s + 1)] × [PID] × [对象]
          ↑
        输出滤波

影响：
  - 闭环极点：可能右移（稳定性下降）
  - 相位裕度：明显下降
  - 剪切频率：下降
```

### 4.3 根轨迹分析

#### 输入滤波
```
s平面
  │         ×（原极点）
  │        ╱
  │       ╱  ← 新极点在-1/τi
  │      ×
  │
  └─────────→ Re

影响：增加一个左半平面极点
      → 阻尼增大，稳定性改善
      → 响应略微变慢
```

#### 输出滤波
```
s平面
  │         ×（原极点）
  │        ╱
  │       ╱  ← 新极点在-1/τo
  │      ×
  │
  └─────────→ Re

影响：增加一个左半平面极点
      → 相位滞后增加
      → 可能失稳（需重新调参）
```

## 5. 配合使用策略

### 5.1 双滤波系统框图

```
          ┌─────────────────────────────────┐
          │                                 │
目标 ──→(+)[求和]──→[PID]──→[滤波2]──→[执行器]──→ 对象
          ──                 ↑                │
                            │                │
                            └────────[滤波1]←┘
                                     ↑
                                  实际转速
```

### 5.2 配合原则

#### 原则1：输入为主，输出为辅
```
策略：
  - 优先使用输入滤波
  - 仅在必要时添加输出滤波

原因：
  - 输入滤波主要改善稳定性
  - 输出滤波降低响应速度
```

#### 原则2：避免双重强滤波
```
错误配置：
  输入滤波：k = 128  ← 很强
  输出滤波：k = 100  ← 很强
  结果：系统响应极慢，可能失稳 ❌

正确配置：
  输入滤波：k = 64   ← 中等
  输出滤波：k = 20   ← 较弱
  结果：平衡性能与稳定性 ✅
```

#### 原则3：总延迟限制
```
总时间常数 ≈ τ输入 + τ输出
         ≤ (1/10) × 系统主导时间常数

示例：
  电机时间常数 = 1s
  总滤波延迟 ≤ 0.1s

  配置：
    输入滤波：τi = 50ms
    输出滤波：τo = 30ms
    总延迟：80ms < 100ms ✅
```

### 5.3 滤波器选择

#### 输入滤波器选择

| 测量类型 | 推荐滤波器 | 滤波系数(k) | 原因 |
|---------|-----------|------------|------|
| **霍尔转速** | 一阶低通 | 40-128 | 噪声大 |
| **编码器转速** | 一阶低通 | 16-40 | 噪声小 |
| **ADC电流** | 一阶低通 + 过采样 | 64-256 | 噪声很大 |
| **压力传感器** | 一阶低通 | 32-64 | 中等噪声 |
| **温度传感器** | 一阶低通 | 100-256 | 慢变化 |

#### 输出滤波器选择

| 执行器类型 | 推荐滤波器 | 滤波系数(k) | 原因 |
|-----------|-----------|------------|------|
| **电机PWM** | 一阶低通 | 10-30 | 平滑启动 |
| **伺服驱动** | 不需要 | - | 内环已滤波 |
| **步进电机** | 弱滤波 | 10-20 | 保持分辨率 |
| **比例阀** | 一阶低通 | 20-40 | 防水锤 |
| **开关阀** | 不适用 | - | 离散控制 |

## 6. 实际案例分析

### 6.1 案例1：纯输入滤波（项目当前配置）

#### 系统描述
```
单片机 + 无刷电机 + 霍尔传感器
控制：速度环（位置式PID）
滤波：仅输入滤波
```

#### 配置代码
```c
// MotorDriver.c:601 - 低速段
*CaptureCnt = FirstOrder_Filter(TimerCapCnt, *CaptureCnt, 40);

// MotorDriver.c:622 - 高速段
*CaptureCnt = FirstOrder_Filter(TimerCapCnt, *CaptureCnt, 128);

// Adc.c:450 - 电流测量
PreValue = FirstOrder_Filter(AvgValue, PreValue, 64);
```

#### 性能分析
```
优点：
  ✅ 转速测量平滑，PID计算稳定
  ✅ 电流采样准确，转矩控制精确
  ✅ 系统响应快，动态性能好

缺点：
  ⚠️ 启动时PWM可能有跳变
  ⚠️ 换相瞬间可能有电流冲击
  ⚠️ 执行器（电机）可能有机械应力
```

### 6.2 案例2：双滤波系统

#### 系统描述
```
高性能伺服系统
控制：双闭环（电流环 + 速度环）
滤波：输入 + 输出双滤波
```

#### 配置示例
```c
// 输入滤波（速度反馈）
Speed_Feedback = Filter(Speed_Measured, Speed_Old, 50);

// 速度PID
Error = Speed_Ref - Speed_Feedback;
PID_Out = PID_Calculate(Error);

// 输出滤波（电流给定）
Iq_Ref = Filter(PID_Out, Iq_Ref_Old, 20);

// 电流环（执行器）
Iq_Actual = Current_Control(Iq_Ref);
```

#### 性能对比

| 指标 | 仅输入滤波 | 双滤波 | 改善 |
|-----|----------|--------|------|
| **转速波动** | ±1.5% | ±1.0% | ✅ 33% |
| **启动电流冲击** | 150% | 110% | ✅ 27% |
| **响应时间** | 100ms | 130ms | ❌ 30% |
| **超调量** | 5% | 3% | ✅ 40% |
| **PWM抖动** | 8% | 3% | ✅ 63% |

**结论**：双滤波牺牲少量响应速度，换来了更好的平滑性

### 6.3 案例3：滤波过度的教训

#### 错误配置
```c
// 输入滤波（过强）
Speed_Filtered = Filter(Speed_Raw, Speed_Old, 256);

// PID输出
PWM_Raw = PID_Calculate(Error);

// 输出滤波（过强）
PWM_Final = Filter(PWM_Raw, PWM_Old, 200);
```

#### 问题表现
```
1. 响应极慢：
   - 目标：0 → 5000 RPM
   - 实际：需要5秒才能稳定 ❌

2. 跟踪误差大：
   - 动态负载时
   - 转速跌落500 RPM
   - 恢复时间3秒 ❌

3. 参数调试困难：
   - PID参数难以整定
   - 系统反应迟钝 ❌
```

#### 修正方案
```c
// 输入滤波（降低强度）
Speed_Filtered = Filter(Speed_Raw, Speed_Old, 64);

// PID输出
PWM_Raw = PID_Calculate(Error);

// 输出滤波（降低强度）
PWM_Final = Filter(PWM_Raw, PWM_Old, 30);
```

**结果**：响应时间从5秒降到0.8秒 ✅

## 7. 设计选择指南

### 7.1 决策流程图

```
是否需要滤波？
  │
  ├─ 测量噪声大？
  │   ├─ 是 → 需要【输入滤波】
  │   └─ 否 → 继续
  │
  ├─ 执行器易抖动？
  │   ├─ 是 → 需要【输出滤波】
  │   └─ 否 → 继续
  │
  ├─ 机械冲击敏感？
  │   ├─ 是 → 需要【输出滤波】
  │   └─ 否 → 可能不需要
  │
选择滤波强度？
  │
  ├─ 输入滤波强度选择
  │   ├─ 编码器 → k=16-40（弱）
  │   ├─ 霍尔   → k=40-128（中）
  │   ├─ ADC    → k=64-256（强）
  │   └─ 温度   → k=100-256（很强）
  │
  └─ 输出滤波强度选择
      ├─ 伺服   → 不需要（内环已处理）
      ├─ 步进   → k=10-20（弱）
      ├─ 电机PWM → k=20-40（中）
      └─ 阀门   → k=50-100（强）

验证总延迟
  │
  └─ τ输入 + τ输出 ≤ 0.1 × τ系统
      ├─ 超标 → 减弱滤波
      └─ 合格 → 完成
```

### 7.2 应用场景推荐

#### 场景1：高动态响应系统
```
应用：无人机、机器人关节
要求：快速响应，精确跟踪

推荐配置：
  输入滤波：k=16-32（弱）
  输出滤波：k=10-20（弱或无）

原因：
  - 优先保证响应速度
  - 使用高精度编码器，噪声小
  - 执行器（电机）响应快
```

#### 场景2：稳定运行系统
```
应用：风扇、泵、输送带
要求：稳定运行，低噪声

推荐配置：
  输入滤波：k=64-128（中）
  输出滤波：不使用

原因：
  - 稳定性优先
  - 测量噪声需抑制
  - 执行器不需要平滑
```

#### 场景3：大惯性系统
```
应用：重型电机、升降机
要求：平滑启动，无机械冲击

推荐配置：
  输入滤波：k=32-64（中弱）
  输出滤波：k=30-50（中）

原因：
  - 避免启动冲击
  - 大惯性掩盖了滤波延迟
  - 保护机械设备
```

#### 场景4：高精度系统
```
应用：CNC机床、精密定位
要求：高精度，低纹波

推荐配置：
  输入滤波：k=40-64（中）
  输出滤波：k=20-30（弱）

原因：
  - 平衡精度与响应
  - 减少位置纹波
  - 使用高分辨率编码器
```

```c
int32_t PWM_Output_Filter(int32_t PWM_Raw)
{
    static int32_t PWM_Old = 0;
    int32_t k = 20;  // 弱滤波

    return (PWM_Raw + (k-1) * PWM_Old) / k;

// 修改PID调用

    PID_Location(&TorquePID, ...)
);
```

## 9. 总结

### 9.1 核心要点

1. **输入滤波和输出滤波作用完全不同**
   - 输入：抑制测量噪声，改善稳定性
   - 输出：平滑执行器动作，减少机械冲击

2. **输入滤波是基础，输出滤波是补充**
   - 大多数系统仅需输入滤波
   - 特殊需求才用输出滤波

3. **两者会引入延迟，需权衡**
   - 总延迟 = 输入延迟 + 输出延迟
   - 限制：< 0.1×系统时间常数

4. **滤波强度选择很重要**
   - 过弱：噪声/抖动无法抑制
   - 过强：响应慢，稳定性差

### 9.2 最佳实践

#### ✅ 推荐做法
```
1. 优先配置输入滤波
   - 根据测量噪声强度选择
   - 霍尔：k=40-128
   - 编码器：k=16-40

2. 谨慎添加输出滤波
   - 仅在执行器抖动时使用
   - 强度应弱于输入滤波
   - 考虑分阶段使用

3. 验证总延迟
   - 计算系统时间常数
   - 确保总延迟在限制内

4. 系统化调试
   - 先调输入滤波
   - 后调输出滤波
   - 整体验证性能
```

#### ❌ 避免的错误
```
1. 双重强滤波
   - 输入k=200 + 输出k=150
   - 系统反应迟钝

2. 忽略执行器特性
   - 快速电机用强滤波
   - 慢速阀门不用滤波

3. 过度依赖滤波
   - 用滤波掩盖参数不当
   - 应该先调好PID

4. 不验证延迟
   - 随意设置滤波参数
   - 导致系统不稳定
```

### 9.3 快速参考表

| 问题 | 解决方案 | 滤波类型 | 推荐参数 |
|-----|---------|---------|---------|
| **转速测量抖动** | 输入滤波 | 一阶低通 | k=40-128 |
| **电流采样噪声** | 输入滤波 | 一阶低通 + 过采样 | k=64-256 |
| **电机启动抖动** | 输出滤波 | 一阶低通 | k=20-40 |
| **PWM输出跳变** | 输出滤波 | 一阶低通 | k=30-50 |
| **机械冲击大** | 输出滤波 | 一阶低通 | k=40-60 |
| **响应速度慢** | 减弱滤波 | - | 降低k值 |
