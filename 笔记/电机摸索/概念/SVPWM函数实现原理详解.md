# SVPWM函数实现原理详解

## 一、基本概念

### 1.1 什么是SVPWM

**SVPWM** (Space Vector PWM) 是一种电机控制调制技术，通过合成电压空间矢量来控制三相逆变器，产生圆形旋转磁场。

### 1.2 核心思想

```
三相逆变器 → 8个基本电压矢量 (V0~V7)
            ↓
       选择相邻2个非零矢量 + 2个零矢量
            ↓
       按时间比例合成 → 任意方向电压矢量
```

## 二、代码结构分析

### 2.1 角度定义

```c
#define VECTOR1  0x0000   // 0°   (0/65536)
#define VECTOR2  0x2AAA   // 60°  (10922/65536)
#define VECTOR3  0x5555   // 120° (21845/65536)
#define VECTOR4  0x8000   // 180° (32768/65536)
#define VECTOR5  0xAAAA   // 240° (43690/65536)
#define VECTOR6  0xD555   // 300° (54613/65536)

// 16位无符号整数表示 0-360°
// 范围: 0 ~ 65535
```

### 2.2 幅值限制

```c
#define VOLTS_LIMIT 28377

// 计算: 32768 × √3/2 ≈ 28377
// √3/2 ≈ 0.866 是线性调制的最大值
// 超过此值会过调制，导致波形失真
```

### 2.3 正弦查表

```c
int sinetable[172] = {0, 201, 401, ..., 28377};

// 为什么是172个点？
// 1024点/周期 × (60°/360°) = 170.67 ≈ 171点
// +1个结束点 = 172个点

// 查表方式: angle >> 6
// angle是16位，右移6位 = 除以64
// 65536 / 64 = 1024 索引范围
// 但只用了前172个 (60°范围)
```

## 三、算法流程详解

### 3.1 主函数结构

```c
void SVM(int volts, unsigned int angle)
{
    // 输入参数:
    // volts: 调制幅值 (0 ~ 28377)
    // angle: 电角度 (0 ~ 65535, 对应0°~360°)

    // 输出全局变量:
    // PDC1, PDC2, PDC3: 三相占空比值
}
```

### 3.2 计算PWM周期

```c
tpwm = SystemCoreClock / FPWM;

// 示例: SystemCoreClock = 96MHz, FPWM = 24000Hz
// tpwm = 96,000,000 / 24,000 = 4000个计数
```

### 3.3 扇区判断

```
         60°    120°
           \    /
    扇区2   \  /  扇区3
      \   6  \/  1  /
    240°--------°0
      /         \
    扇区5       扇区1
      /         \
```

**判断逻辑**:

```c
if (angle < VECTOR4)          // 0° ~ 180°
    if (angle < VECTOR3)      // 0° ~ 120°
        if (angle < VECTOR2)  // 0° ~ 60°  → 扇区1
        else                  // 60°~120° → 扇区2
    else                      // 120°~180°→ 扇区3
else                          // 180°~360°
    if (angle < VECTOR6)      // 180°~300°
        if (angle < VECTOR5)  // 180°~240°→ 扇区4
        else                  // 240°~300°→ 扇区5
    else                      // 300°~360°→ 扇区6
```

### 3.4 角度归一化

```c
// 示例: angle = 0x4000 (90°), 落在扇区2
angle2 = angle - VECTOR2;      // 0x4000 - 0x2AAA = 0x1556 (30°)
angle1 = SIXTY_DEG - angle2;   // 0x2AAA - 0x1556 = 0x1554 (30°)

// angle1 + angle2 = 60°
```

### 3.5 时间计算

```c
// 从正弦表查值
t1 = sinetable[angle1 >> 6];  // sin(60°-θ)
t2 = sinetable[angle2 >> 6];  // sin(θ)

// 幅值缩放
voltstpwm = volts * tpwm / 28377;
t1 = t1 * voltstpwm / 28377;
t2 = t2 * voltstpwm / 28377;

// 零矢量时间 (半周期)
half_t0 = (tpwm - t1 - t2) >> 1;
```

**公式推导**:

```
T1 = (Tpwm × Vref × sin(60°-θ)) / (√3/2 × Vmax)
T2 = (Tpwm × Vref × sin(θ)) / (√3/2 × Vmax)
T0 = Tpwm - T1 - T2
```

## 四、占空比计算（各扇区）

### 4.1 扇区1 (0°~60°)

```c
PDC1 = t1 + t2 + half_t0;  // 最大
PDC2 = t2 + half_t0;
PDC3 = half_t0;             // 最小
```

时序: `t0/2 → t1 → t2 → t0 → t2 → t1 → t0/2`

### 4.2 扇区2 (60°~120°)

```c
PDC1 = t1 + half_t0;
PDC2 = t1 + t2 + half_t0;  // 最大
PDC3 = half_t0;
```

### 4.3 扇区3 (120°~180°)

```c
PDC1 = half_t0;
PDC2 = t1 + t2 + half_t0;  // 最大
PDC3 = t2 + half_t0;
```

### 4.4 扇区4 (180°~240°)

```c
PDC1 = half_t0;
PDC2 = t1 + half_t0;
PDC3 = t1 + t2 + half_t0;  // 最大
```

### 4.5 扇区5 (240°~300°)

```c
PDC1 = t2 + half_t0;
PDC2 = half_t0;
PDC3 = t1 + t2 + half_t0;  // 最大
```

### 4.6 扇区6 (300°~360°)

```c
PDC1 = t1 + t2 + half_t0;  // 最大
PDC2 = half_t0;
PDC3 = t1 + half_t0;
```

## 五、波形示例

假设 `volts = 20000, angle = 0x1555 (30°)`

```
扇区1:
angle2 = 0x1555 - 0x0000 = 0x1555 (30°)
angle1 = 0x2AAA - 0x1555 = 0x1555 (30°)

t1 = sinetable[0x1555 >> 6] = sinetable[85] ≈ 14189
t2 = sinetable[0x1555 >> 6] = sinetable[85] ≈ 14189

voltstpwm = 20000 × 4000 / 28377 ≈ 2820
t1 = t2 = 14189 × 2820 / 28377 ≈ 1407
half_t0 = (4000 - 1407 - 1407) / 2 = 593

PDC1 = 1407 + 1407 + 593 = 3407  (85.2%)
PDC2 = 1407 + 593 = 2000        (50.0%)
PDC3 = 593                     (14.8%)
```

## 六、七段式开关序列

```
        PDC1    |    PDC2    |    PDC3
扇区1:  100%  →  50%  →  15%  →  15%  →  50%  →  85%  →  100%
        V0     V4    V6     V7     V6     V4     V0

        ↑7段式SVPWM开关序列
```

## 七、优化要点

1. **查表法**: 避免实时计算sin/cos，提高速度
2. **对称性**: 只存储60°正弦表，利用对称性
3. **定点运算**: 使用整数运算，避免浮点开销
4. **扇区判断**: 快速判断算法，减少分支

## 八、总结

```
SVM函数本质:
    输入: 幅值 + 电角度
    ↓
    扇区判断 → 角度归一化 → 查表 → 时间计算 → 占空比分配
    ↓
    输出: 三相PWM占空比
```

这个SVPWM实现采用了**七段式对称PWM**，具有：

- 低谐波含量
- 开关损耗均衡
- 直流电压利用率高（可达1.0，相比SPWM的0.866）
- 适合BLDC/PMSM电机的高性能控制
