# PWM与转速力矩关系详解

## 一、核心关系链

### 1.1 基本物理关系

```
PWM占空比 → 平均电压 → 电机电流 → 电磁力矩 → 转速
                ↑                         ↓
                └───────── 反电动势 ──────┘
```

### 1.2 简化关系式

```
力矩（T）∝ 电流（I）
电流（I）∝ （电压 - 反电动势）/ 电阻
电压（U）∝ PWM占空比 × 母线电压
反电动势 ∝ 转速

结论：
1. 空载时：转速 ∝ PWM占空比
2. 负载时：力矩 ∝ PWM占空比（转速恒定条件下）
```

## 二、PWM占空比的作用

### 2.1 PWM调压原理

```
母线电压：24V（恒定）

PWM占空比50%：
  平均电压 = 24V × 50% = 12V

PWM占空比75%：
  平均电压 = 24V × 75% = 18V

PWM占空比100%：
  平均电压 = 24V × 100% = 24V
```

### 2.2 PWM对转速的影响（空载）

```
空载条件下（无负载力矩）：

PWM 50% → 12V → 转速 1500 RPM
PWM 70% → 16.8V → 转速 2100 RPM
PWM 90% → 21.6V → 转速 2700 RPM

关系：转速与PWM占空比成正比
```

### 2.3 PWM对力矩的影响（负载）

```
恒速条件下（转速闭环控制）：

转速 = 3000 RPM（恒定）

负载1Nm：
  PWM 70% → 电流3A → 力矩1Nm

负载2Nm：
  PWM 85% → 电流6A → 力矩2Nm

负载3Nm：
  PWM 95% → 电流9A → 力矩3Nm

关系：力矩与PWM占空比成正比（转速恒定时）
```

## 三、PID闭环控制原理

### 3.1 控制框图

```
         用户设定
            ↓
    ┌──────────────┐
    │ 目标转速      │ 3000 RPM（固定）
    └──────────────┘
            ↓
    ┌──────────────────────────────────┐
    │         PID控制器                 │
    │                                  │
    │  误差 = 目标 - 实际                │
    │  PWM = Kp×误差 + Ki×积分 + Kd×微分 │
    └──────────────────────────────────┘
            ↓
        PWM输出（可变）
            ↓
    ┌──────────────┐         ┌──────────────┐
    │   三相桥     │────────→│   BLDC电机   │
    │  (MCPWM)     │ PWM     │              │
    └──────────────┘         └──────────────┘
                                    ↓
                            ┌──────────────┐
                            │  霍尔传感器   │
                            └──────────────┘
                                    ↓
                              实际转速（反馈）
                                    ↑
                                    └── 反馈给PID
```

### 3.2 PID三项作用

| 参数 | 作用 | 对PWM的影响 |
|------|------|------------|
| **Kp（比例）** | 快速响应误差 | 误差越大，PWM变化越大 |
| **Ki（积分）** | 消除稳态误差 | 长期有误差，PWM持续调整 |
| **Kd（微分）** | 减少超调 | 误差变化快，PWM提前调整 |

## 四、定速控制动态过程

### 4.1 启动过程

```
初始状态：
目标转速 = 3000 RPM
实际转速 = 0 RPM
PWM占空比 = 0%

第1周期（t=10ms）：
误差 = 3000 - 0 = 3000 RPM
PID计算：PWM = 50%
实际转速：0 → 800 RPM

第2周期（t=20ms）：
误差 = 3000 - 800 = 2200 RPM
PID计算：PWM = 70%
实际转速：800 → 1500 RPM

第3周期（t=30ms）：
误差 = 3000 - 1500 = 1500 RPM
PID计算：PWM = 85%
实际转速：1500 → 2300 RPM

第4周期（t=40ms）：
误差 = 3000 - 2300 = 700 RPM
PID计算：PWM = 90%
实际转速：2300 → 2800 RPM

第5周期（t=50ms）：
误差 = 3000 - 2800 = 200 RPM
PID计算：PWM = 88%（开始回调）
实际转速：2800 → 3050 RPM（轻微超调）

第6周期（t=60ms）：
误差 = 3000 - 3050 = -50 RPM（负误差）
PID计算：PWM = 87%
实际转速：3050 → 3000 RPM

稳定状态：
误差 = 3000 - 3000 = 0 RPM
PWM = 85%（稳定在某个值）
实际转速 = 3000 RPM ✓
```

### 4.2 负载突变过程

```
初始稳定状态：
目标转速 = 3000 RPM
实际转速 = 3000 RPM
PWM占空比 = 70%
负载力矩 = 1.0 Nm
电机电流 = 3A

t=100ms：负载突然增大到 2.0 Nm
    ↓
转速下降：3000 → 2700 RPM（惯性）
    ↓
PID检测误差：3000 - 2700 = 300 RPM
    ↓
PID增加PWM：70% → 85%
    ↓
电流增大：3A → 6A
    ↓
力矩增大：1.0 Nm → 2.0 Nm
    ↓
转速恢复：2700 → 3000 RPM（约200ms后）

新的稳定状态：
目标转速 = 3000 RPM ✓
实际转速 = 3000 RPM ✓
PWM占空比 = 85%（增加了15%）
负载力矩 = 2.0 Nm
电机电流 = 6A
```

### 4.3 负载减小过程

```
初始状态（重载）：
目标转速 = 3000 RPM
实际转速 = 3000 RPM
PWM占空比 = 90%
负载力矩 = 2.5 Nm
电机电流 = 7.5A

t时刻：负载减小到 1.0 Nm
    ↓
转速上升：3000 → 3300 RPM（力矩过剩）
    ↓
PID检测误差：3000 - 3300 = -300 RPM（负误差）
    ↓
PID减小PWM：90% → 72%
    ↓
电流减小：7.5A → 3A
    ↓
力矩减小：2.5 Nm → 1.0 Nm
    ↓
转速恢复：3300 → 3000 RPM

新的稳定状态：
目标转速 = 3000 RPM ✓
实际转速 = 3000 RPM ✓
PWM占空比 = 72%（降低了18%）
负载力矩 = 1.0 Nm
电机电流 = 3A
```

## 五、实际代码分析

### 5.1 项目中的PID实现

```c
// MotorDriver.c

// PID结构体
typedef struct
{
    int32_t LastError;    // 上次误差
    int32_t SumError;     // 误差积分
    int32_t Kp;           // 比例系数
    int32_t Ki;           // 积分系数
    int32_t Kd;           // 微分系数
} PID_TypeDef;

// PID计算函数
int32_t PID_Location(PID_TypeDef *PID, int32_t RefRPM, int32_t CurRPM, int16_t PWM_Duty_Max)
{
    int32_t iError, Duty;

    // 1. 计算误差
    iError = RefRPM - CurRPM;

    // 2. 积分项（累加误差）
    PID->SumError += iError;

    // 3. 限制积分范围（防止积分饱和）
    if(PID->SumError > 10000) PID->SumError = 10000;
    if(PID->SumError < -10000) PID->SumError = -10000;

    // 4. PID计算
    Duty = (PID->Kp * iError)                       // 比例项
         + (PID->Ki * PID->SumError)                // 积分项
         + (PID->Kd * (iError - PID->LastError));  // 微分项

    // 5. 更新上次误差
    PID->LastError = iError;

    // 6. 限幅
    if(Duty < 0) Duty = 0;
    if(Duty > PWM_Duty_Max) Duty = PWM_Duty_Max;

    return Duty;
}
```

### 5.2 主循环中的调用

```c
// main.c

void main_loop(void)
{
    while(1)
    {
        // 1. 读取目标转速（外部输入，用户设定）
        Motor_Speed.RefRPM = Measure_ADJ_Input(PWM_STOP_DUTY);
        // 假设：RefRPM = 3000 RPM

        // 2. 读取实际转速（霍尔传感器反馈）
        Motor_Speed.CurRPM = Measure_Motor_RPM();
        // 当前：CurRPM = 2800 RPM

        // 3. PID计算（自动调节PWM）
        BLDC_Motor.PWM_Duty = PID_Location(&sPID,
            Motor_Speed.RefRPM,          // 3000（目标）
            Motor_Speed.CurRPM,          // 2800（实际）
            BLDC_Motor.OCP_PWMDutyLimit  // 1000（最大限制）
        );
        // PID输出：PWM_Duty = 850（85%）

        // 4. 应用PWM输出
        MCPWM_SetDuty(BLDC_Motor.PWM_Duty);
        // 三相桥输出：85%占空比

        // 5. 看门狗喂狗
        IWDG_ReloadCounter();
    }
}
```

## 七、PWM、转速、力矩三者关系总结

### 7.1 关系矩阵

| 控制方式 | 固定变量 | 调节变量 | 结果变量 |
|---------|---------|---------|---------|
| **开环调速** | PWM | - | 转速（随负载变化） |
| **恒速控制** | 转速 | PWM | 力矩（随负载自动变化） |
| **恒力矩控制** | 电流(PWM) | - | 转速（随负载变化） |

### 7.2 核心结论

```
1. PWM是控制手段，不是控制目标
   ├─ 开环：PWM直接决定转速
   └─ 闭环：PWM自动调节以维持目标转速

2. 力矩是被动变量
   ├─ 定速控制：力矩自动匹配负载
   └─ 力矩控制：转速自动变化

3. 转速可以是目标或结果
   ├─ 定速控制：转速是目标（闭环）
   └─ 力矩控制：转速是结果（开环）

4. PID的作用
   ├─ 检测转速误差
   ├─ 自动调节PWM
   └─ 维持转速恒定
```

## 八、常见应用场景

### 8.1 场景1：恒速运行（如风扇）

```
目标：无论负载如何，保持3000 RPM

实现：
- 设定目标转速3000 RPM
- PID自动调节PWM
- 负载变化时PWM自动补偿

表现：
- 空载：PWM 60%
- 半载：PWM 75%
- 满载：PWM 90%
- 转速始终：3000 RPM ✓
```

### 8.2 场景2：调速运行（如电动工具）

```
目标：用户设定不同转速

实现：
- 用户调整电位器
- 改变目标转速
- PID自动跟随

表现：
- 低速：目标1000 RPM → PWM 40%
- 中速：目标2000 RPM → PWM 60%
- 高速：目标4000 RPM → PWM 90%
```

### 8.3 场景3：负载自适应（如电动汽车）

```
目标：保持车速，适应坡度

实现：
- 设定目标车速（对应电机转速）
- 上坡时负载增大
- PID自动增加PWM
- 下坡时负载减小
- PID自动减小PWM

表现：
- 平路：PWM 60%，电流3A
- 上坡：PWM 85%，电流6A
- 下坡：PWM 40%，电流1A
- 车速始终恒定 ✓
```

### 9.1 如何观察PWM调节

```c
// Ref:3000 Cur:2800 PWM:850 Cur:3000mA
// Ref:3000 Cur:2950 PWM:830 Cur:2800mA
// Ref:3000 Cur:3000 PWM:820 Cur:2500mA（稳定）
```

### 9.2 如何判断PID是否正常

```
正常情况：
├─ 启动时PWM逐步增加
├─ 接近目标转速时PWM稳定
├─ 负载变化时PWM自动调整
└─ 稳定后PWM在某个值附近小幅波动

异常情况：
├─ PWM震荡（忽大忽小）→ Kp过大
├─ PWM缓慢变化（响应慢）→ Kp过小
├─ 转速无法稳定在目标值 → Ki不足
└─ 超调过大（转速过冲）→ Kd不足
```

### 9.3 PID参数调整方法

```c
// 初始参数
sPID.Kp = 1.0;
sPID.Ki = 0.1;
sPID.Kd = 0.01;

// 调整步骤：

// 1. 先调Kp
//    Kp = 0.5 → 响应慢，无超调
//    Kp = 1.0 → 响应适中
//    Kp = 2.0 → 响应快，有超调

// 2. 再调Ki
//    Ki = 0.05 → 有稳态误差
//    Ki = 0.1  → 误差小
//    Ki = 0.2  → 无误差，但易震荡

// 3. 最后调Kd
//    Kd = 0.0   → 超调大
//    Kd = 0.01  → 超调小
//    Kd = 0.05  → 无超调，但响应慢
```

## 十、关键要点总结

### 10.1 核心概念

```
✅ PWM是控制手段，不是控制目标
✅ 定速控制通过PID自动调节PWM
✅ 力矩随负载变化，PWM自动匹配
✅ 转速误差驱动PWM变化
✅ 闭环控制比开环控制更稳定
```

### 10.2 记忆口诀

```
PWM调电压，电压生电流，
电流生力矩，力矩转转速，
PID看误差，自动调PWM，
转速恒定了，力矩随负载。
```

### 10.3 实际应用建议

```
开发阶段：
1. 先开环测试PWM与转速关系
2. 再加入PID闭环控制
3. 逐步调整PID参数
4. 测试不同负载下的表现

生产阶段：
1. 根据负载类型选择PID参数
2. 保留参数调节接口
3. 添加PWM限幅保护
4. 监控电流防止过载
```
