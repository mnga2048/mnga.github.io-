# BLDC电机三相桥采样电路原理

## 一、三相桥驱动与电机电流电压关系

### 1.1 三相桥基本工作原理

#### 三相桥拓扑结构

```
        VDD（母线电压）
           │
    ┌──────┴──────┐
    │             │
  ┌─┴─┐         ┌─┴─┐
  │ Q1│         │ Q4│
  └─┬─┘         └─┬─┘
    │ U相          │ W相
    │             │
    │    ┌───┐    │
    │    │ M │    │   电机
    │    │ O │    │   三相绕组
    │    │ T │    │
    │    └───┘    │
    │             │
  ┌─┴─┐         ┌─┴─┐
  │ Q3│         │ Q6│
  └───┘         └───┘
    │ V相          │
    │             │
  GND（地）
```

#### 开关状态与电流路径

**三相桥通过6个MOSFET（Q1-Q6）控制三相电流**

每个时刻，三相桥的工作状态：
- **上桥臂导通1相**（Q1或Q3或Q5）
- **下桥臂导通1相**（Q4或Q6或Q2）
- **第3相关断**

示例：
```
状态1：Q1导通（U+），Q6导通（W-）
电流路径：VDD → Q1 → U相绕组 → W相绕组 → Q6 → GND

状态2：Q1导通（U+），Q2导通（V-）
电流路径：VDD → Q1 → U相绕组 → V相绕组 → Q2 → GND

状态3：Q3导通（V+），Q2导通（V-）
电流路径：VDD → Q3 → V相绕组 → U相绕组 → Q2 → GND

... 共6个状态（对应6步换相）
```

---

### 1.2 PWM调制下的电流电压波形

#### PWM调制原理

```
PWM频率：20kHz（周期50μs）
一个PWM周期内：

0-25μs（50%占空比）：
  Q1导通，Q6导通
  → U相通电，W相通电
  → 电流从U流向W
  → 电机产生转矩

25-50μs（关断时间）：
  Q1关断或Q6关断
  → 电流通过续流二极管/死区时间
  → 电流逐渐减小
```

#### 相电流波形

```
相电流（理想方波）：
  Iu
  ↑
  │     ┌───┐     ┌───┐
  │     │   │     │   │
  │ ┌───┘   └───┘   └───┘
  │
  └─────────────────────→ t
  60°  120° 180° 240°

实际相电流（带纹波）：
  Iu
  ↑
  │     ╭───╮     ╭───╮
  │     │   │     │   │
  │ ╭───╯   ╰───╯   ╰───╯
  │   ↑ 纹波
  └─────────────────────→ t
```

#### 母线电流波形

```
母线电流（Ibus）：

Ibus
  ↑
  │     ┌─┐ ┌─┐ ┌─┐ ┌─┐
  │     │ │ │ │ │ │ │ │ │
  │   ┌─┘ └─┘ └─┘ └─┘ └─┘
  │   │
  │   │← 换相时刻
  └───┴───────────────→ t

特点：
1. 不是正弦波
2. 是脉冲序列
3. 换相时电流突变
4. PWM开关产生高频纹波
```

### 1.3 电机电压

#### 相电压（对地）

```
U相电压（Vu）：
 Vu
  ↑
  │    ┌────┐    ┌────┐
  │    │    │    │    │
  └────┘    └────┘    └──→ t
  120°  180° 240°

幅值：
- 导通时：VDD/2（中点电压）
- 关断时：感应电压
```

#### 线电压（相间）

```
线电压（Uuv = Vu - Vv）：

Uuv
  ↑
  │     ┌──────┐
  │     │      │
  │   ┌─┘      └─┐
  │   │          │
  └───┘          └──→ t

幅值：
- 导通时：VDD（全母线电压）
- 关断时：0V或感应电压
```

## 二、电流采样电路设计

### 2.1 电流采样方法选择

#### 方法1：母线电流采样（推荐）⭐

```
采样位置：母线负极（GND回流）

        VDD
         │
    ┌────┴────┐
    │ 三相桥   │
    │         │
    └────┬────┘
         │
       ┌─┴─┐
       │Rs │ ← 采样电阻
       └─┬─┘
         │
         ↓
    ┌─────┴─────┐
    │  运放电路  │
    │（差分放大）│
    └─────┬─────┘
         │
         ↓
        ADC
```

**优点：**
- ✅ 只需1个采样电阻
- ✅ 电路简单
- ✅ 成本低
- ✅ 适用于过流保护

**缺点：**
- ❌ 只能测量总电流
- ❌ 不能分相测量
- ❌ 换相时刻有干扰

**计算公式：**
```
Ibus = (Vadc - Voffset) / (Rs × Gain)

其中：
- Vadc：ADC采样电压
- Voffset：运放零点偏置（通常是VDD/2）
- Rs：采样电阻阻值
- Gain：运放增益
```

---

#### 方法2：相电流采样（两电阻法）

```
采样位置：两相下桥臂下方

        VDD
         │
    ┌────┴────┐
    │ 三相桥   │
    └┬───┬───┬┘
      │   │   │
     Q4  Q6  Q2
      │   │   │
     Rs1 Rs2
      │   │
     ┌─┴─┐┌┴─┐
     │运放││运放│
     └─┬─┘└┬─┘
       │    │
       └──┬─┘
         ↓
        ADC（多路复用）
```

**优点：**
- ✅ 可以测量2相电流
- ✅ 可用于FOC（磁场定向控制）

**缺点：**
- ❌ 需要2个采样电阻
- ❌ 需要2个运放
- ❌ PWM开关时干扰大
- ❌ 需要同步采样

**计算公式：**
```
Iu = (Vadc1 - Voffset) / (Rs1 × Gain)
Iv = (Vadc2 - Voffset) / (Rs2 × Gain)

第三相电流：
Iw = -(Iu + Iv)  （基尔霍夫电流定律）
```

---

### 2.2 采样电阻选型

#### 采样电阻的作用

```
1. 将电流转换为电压（V = I × R）
2. 提供电流检测信号
3. 用于过流保护
```

#### 选型参数

**1. 阻值选择**

```
原则：
- 阻值太小 → 信号弱，ADC精度不够
- 阻值太大 → 功耗大，压降大

计算公式：
Rs ≤ Vadc_max / Ibus_max × η

其中：
- Vadc_max：ADC最大输入电压（如3.3V）
- Ibus_max：最大母线电流（如10A）
- η：利用率（取0.8，留20%裕度）

示例：
Rs ≤ 3.3V / 10A × 0.8
   ≤ 0.264Ω

选择：Rs = 0.1Ω（标准值）
```

**2. 功率选择**

```
功耗计算：
P = I² × Rs

最大电流10A时：
P = 10² × 0.1 = 10W

选择电阻功率：
P_resistor ≥ P × 安全系数
              ≥ 10W × 2
              ≥ 20W

实际选择：20W或25W电阻（水泥电阻/分流器）
```

**3. 精度选择**

```
精度要求：
- 温度系数：≤50ppm/°C（低温漂）
- 精度：±1%或更好
- 封装：适合PCB安装（2512或更大）
```

---

### 2.3 运放电路设计

#### 为什么需要运放？

```
原因1：信号放大
采样电阻电压小：
- 0.1Ω × 10A = 1V
- 需要放大到ADC可测范围

原因2：偏置调整
ADC只能测正电压：
- 双向电流需要偏置到VDD/2
- 静态时输出VDD/2（1.65V）

原因3：阻抗匹配
运放高输入阻抗：
- 不影响采样电路
- 驱动能力强
```

#### 运放配置方案

**方案1：单向电流放大**

```
电路结构：
    Ibus → Rs → 运放(同相) → ADC

适用场景：
- 只测量单向电流
- 不需要双向测量

特点：
- 简单
- 成本低
- 精度较高
```

**方案2：双向电流放大（差分）** ⭐

```
电路结构：
         VDD
          │
         ─┴─
          │
    ┌─────┴─────┐
    │   Rs      │
    │  采样电阻  │
    └─────┬─────┘
          │
    ┌─────┴─────┐
    │  运放电路  │
    │（差分放大）│
    └─────┬─────┘
          │
         Vout
         ↕ ADC

适用场景：
- 双向电流测量
- 需要检测充放电

特点：
- 可测正负电流
- 零点在VDD/2
- 复杂度高
```

#### 运放参数计算

```
增益计算：
Gain = Rf / Ri

目标：
- 最大电流10A → ADC输出3.3V
- 采样电压：10A × 0.1Ω = 1V
- 需要放大：3.3倍

选择：
Ri = 1kΩ
Rf = 3.3kΩ
Gain = 3.3kΩ / 1kΩ = 3.3 ✓

偏置电压：
Voffset = VDD / 2 = 1.65V

输出电压范围：
- 电流-10A：0V（被限幅）
- 电流0A：1.65V（偏置点）
- 电流+10A：3.3V（满量程）
```

## 三、电压采样电路设计

### 3.1 母线电压采样

#### 采样原理

```
直接分压法：

    Vbus（0-40V）
        │
       ─┴─
        │
    ┌───┴───┐
    │  R1   │  上分压电阻
    └───┬───┘
        ├─── Vdiv（分压点）
        │
    ┌───┴───┐
    │  R2   │  下分压电阻
    └───┬───┘
        │
       GND

分压公式：
Vdiv = Vbus × R2 / (R1 + R2)
```

#### 电阻选型

```
设计要求：
- Vbus_max = 40V
- ADC输入 = 0-3.3V
- 分压比 = 3.3 / 40 = 0.0825

计算：
R2 / (R1 + R2) = 0.0825

选择：
R1 = 100kΩ
R2 = 9.1kΩ

验证：
分压比 = 9.1k / (100k + 9.1k) = 0.083 ✓
40V输入时：
Vdiv = 40 × 0.083 = 3.32V ≈ 3.3V ✓
```

#### 功耗计算

```
R1功耗：
P1 = V² / R = 40² / 100k = 16mW

R2功耗：
P2 = (3.3)² / 9.1k = 1.2mW

总功耗：17.2mW（可忽略）

电阻封装：0603或0805（1/16W足够）
```

---

### 3.2 相电压采样（可选）

```
电阻分压网络：

         Vu
          │
         ─┴─
          │
       ┌─┴─┐
       │R1 │
       └─┬─┘
         ├─── Vdiv
       ┌─┴─┐
       │R2 │
       └─┬─┘
         │
        GND

注意：
1. 需要三相分别采样（3个分压电路）
2. 共地参考点
3. 需要考虑开关噪声
```

## 四、ADC配置原理

### 4.1 ADC基本参数

```
分辨率：
- 12位ADC：0-4095（常用）
- 14位ADC：0-16383（高精度）
- 16位ADC：0-65535（精密）

采样率：
- 10kSPS：低速（慢速变化）
- 100kSPS：中速（一般应用）
- 1MSPS：高速（高频采样）

输入范围：
- 0-3.3V（单端）
- 0-Vref（差分）
```

---

### 4.2 ADC采样模式

#### 单次采样

```
特点：
- 触发一次采一次
- 适合慢速变化信号
- 功耗低

流程：
启动转换 → 等待完成 → 读取结果

应用：
- 温度采样
- 电压监测（慢速）
```

#### 连续采样

```
特点：
- 自动连续采样
- 适合快速变化信号
- 功耗较高

流程：
启动 → 采样1 → 采样2 → 采样3 → ...

应用：
- 电流采样（需要实时）
- PWM同步采样
```

#### 扫描模式

```
特点：
- 多通道自动切换
- 依次采样各通道

流程：
通道1 → 通道2 → 通道3 → ... → 通道1

应用：
- 多路电压监测
- 电流+电压同时采集
```

---

### 4.3 ADC时序配置

#### 采样时间

```
采样时间 = 采样保持周期 + 转换时间

采样保持周期：
- 作用：对输入信号充电
- 要求：足够长以稳定信号
- 计算：Tsample ≥ Rin × Csample × ln(2^N)

其中：
- Rin：信号源内阻
- Csample：采样电容
- N：ADC位数（如12位）

示例：
Tsample ≥ 10kΩ × 10pF × ln(4096)
        ≥ 10kΩ × 10pF × 8.3
        ≥ 0.83μs

选择：Tsample = 2μs（安全裕度）
```

#### 转换时间

```
转换时间 = 转换周期 × ADC时钟

示例：
- ADC时钟：1MHz
- 12位ADC需要13个时钟
- 转换时间 = 13 / 1MHz = 13μs
```

#### 总采样周期

```
总周期 = 采样时间 + 转换时间
       = 2μs + 13μs
       = 15μs

采样率 = 1 / 15μs = 66.7kSPS
```

## 五、运放与ADC的配合

### 5.1 为什么要运放？

```
直接ADC采样的问题：
1. 采样电阻电压小（如1V）
2. ADC分辨率浪费
3. 信号弱，易受干扰

运放的作用：
1. 信号放大（提高分辨率）
2. 阻抗匹配（减少负载效应）
3. 偏置调整（双向测量）
4. 滤波降噪（提高精度）
```

---

### 5.2 运放配置要点

#### 输入阻抗

```
要求：
运放输入阻抗 >> 信号源内阻

示例：
信号源内阻：Rs = 0.1Ω
运放输入阻抗：Zin = 1MΩ

负载效应：
误差 = Rs / Zin = 0.1 / 1M = 0.00001%（可忽略）
```

#### 输出阻抗

```
要求：
运放输出阻抗 << ADC输入阻抗

示例：
运放输出阻抗：Zout = 10Ω
ADC输入阻抗：Zadc = 10kΩ

负载效应：
误差 = Zout / Zadc = 10 / 10k = 0.1%（可接受）
```

#### 带宽要求

```
要求：
运放带宽 >> 信号带宽

信号带宽：
- PWM频率：20kHz
- 需要测量：直流到PWM频率的5-10倍
- 所需带宽：100-200kHz

选择：
运放带宽：≥1MHz（足够）
```

---

### 5.3 滤波电路

```
RC低通滤波：

    Vout ────┬──── Vfiltered
             │
            ─┴─ C
             │
            GND

截止频率：
fc = 1 / (2π × R × C)

设计目标：
- 保留直流分量
- 滤除PWM开关噪声
- 截止频率：1-5kHz

选择：
R = 1kΩ
C = 100nF

fc = 1 / (2π × 1k × 100n)
  = 1.59kHz ✓
```

## 六、完整采样流程

### 6.1 电流采样流程

```
1. 物理层：电流流过采样电阻
   Ibus → Rs → 产生压降

2. 模拟层：运放调理信号
   压降 → 差分放大 → 偏置 → 滤波

3. 数字层：ADC转换
   模拟电压 → ADC → 数字值

4. 计算层：软件处理
   ADC值 → 校准 → 滤波 → 电流值

5. 应用层：控制决策
   电流值 → 保护/PID控制
```

---

### 6.2 电压采样流程

```
1. 物理层：高压分压
   Vbus(0-40V) → R1+R2分压 → Vdiv(0-3.3V)

2. 滤波层：RC滤波
   Vdiv → RC滤波 → 消除噪声

3. 数字层：ADC转换
   模拟电压 → ADC → 数字值

4. 计算层：软件换算
   ADC值 → 反算公式 → 实际电压

5. 应用层：保护决策
   电压值 → 过压/欠压判断
```

---

## 七、关键计算公式

### 7.1 电流计算

```
母线电流：
Ibus = (Vadc - Voffset) / (Rs × Gain)

其中：
- Vadc：ADC采样值对应的电压
- Voffset：零点偏置电压（VDD/2）
- Rs：采样电阻（0.1Ω）
- Gain：运放增益（3.3）

示例：
Vadc = 2.5V
Voffset = 1.65V
Rs = 0.1Ω
Gain = 3.3

Ibus = (2.5 - 1.65) / (0.1 × 3.3)
     = 0.85 / 0.33
     = 2.58A
```

---

### 7.2 电压计算

```
母线电压：
Vbus = (Vadc × (R1 + R2)) / R2

其中：
- Vadc：ADC采样值
- R1：上分压电阻（100kΩ）
- R2：下分压电阻（9.1kΩ）

示例：
Vadc = 1.65V（ADC读数2048）
R1 = 100kΩ
R2 = 9.1kΩ

Vbus = (1.65 × (100k + 9.1k)) / 9.1k
     = (1.65 × 109.1k) / 9.1k
     = 180.015 / 9.1
     = 19.78V
```

---

### 7.3 功率计算

```
电机功率：
P = Vbus × Ibus × η

其中：
- Vbus：母线电压
- Ibus：母线电流
- η：电机效率（0.8-0.9）

示例：
Vbus = 24V
Ibus = 5A
η = 0.85

P = 24 × 5 × 0.85 = 102W
```

## 八、选型指南

### 8.1 采样电阻选型

```
1. 确定最大电流
   Imax = 10A

2. 计算阻值
   Rs ≤ Vadc_max / Imax
      ≤ 3.3V / 10A
      ≤ 0.33Ω

3. 选择标准值
   Rs = 0.1Ω

4. 验证功率
   P = I² × R = 10² × 0.1 = 10W
   选择20W电阻

5. 确定精度
   精度：±1%
   温漂：≤50ppm/°C
```

---

### 8.2 运放选型

```
1. 确定电源电压
   VDD = 3.3V（单电源）

2. 确定带宽
   所需带宽：≥200kHz
   选择：1MHz以上

3. 确定输入阻抗
   Zin ≥ 1MΩ

4. 确定输出驱动
   Iout ≥ 10mA

5. 确定精度
   失调电压：<1mV
   温漂：<5μV/°C

6. 推荐型号类型
   - 零漂运放
   - 轨到轨运放
   - 低噪声运放
```

---

### 8.3 ADC配置要点

```
1. 分辨率选择
   - 12位：一般应用（4096级）
   - 14位：高精度（16384级）
   - 16位：精密测量（65536级）

2. 采样率选择
   - 电流采样：≥100kSPS（实时）
   - 电压采样：≥10kSPS（慢速）

3. 输入范围
   - 单端：0-3.3V
   - 差分：±Vref/2

4. 触发方式
   - 定时器触发（PWM同步）
   - 软件触发（按需采样）

5. DMA支持
   - 高速采样时使用DMA
   - 减轻CPU负担
```

## 九、常见问题

### Q1：为什么采样电阻要小？

```
A: 减小功耗和压降

功耗：P = I² × R
- 阻值大 → 功耗大 → 发热严重

压降：V = I × R
- 阻值大 → 压降大 → 母线电压降低

选择原则：
- 在保证测量精度的前提下越小越好
- 一般取0.01-0.1Ω
```

---

### Q2：为什么需要运放？

```
A: 信号调理和阻抗匹配

1. 放大微弱信号
   采样电压：1V
   放大后：3.3V（充分利用ADC范围）

2. 提高测量精度
   ADC分辨率：12位（4096级）
   直接采样：3.3V / 4096 = 0.8mV/级
   放大后：0.8mV / 3.3 = 0.24mV/级

3. 阻抗匹配
   信号源内阻：0.1Ω
   运放输入阻抗：1MΩ
   负载效应：可忽略
```

---

### Q3：ADC采样率要多快？

```
A: 根据信号带宽决定

Nyquist采样定理：
fs ≥ 2 × fmax

实际应用：
- 直流信号：≥100SPS（慢）
- PWM电流：≥100kSPS（快）
- 电压监测：≥1kSPS（中）

考虑因素：
1. 信号变化速度
2. PWM频率（20kHz）
3. 需要的响应速度
```

---

### Q4：如何提高采样精度？

```
方法1：硬件滤波
- RC滤波
- π型滤波
- 差分传输

方法2：软件滤波
- 滑动平均
- 一阶滤波
- 中值滤波

方法3：校准
- 零点校准
- 增益校准
- 温度补偿

方法4：采样策略
- PWM同步采样
- 多次平均
- 过采样
```

## 十一、实际应用示例

### 11.1 完整电流采样链路

```
物理层：
Ibus = 5A → Rs(0.1Ω) → Vdrop = 0.5V

模拟层：
0.5V → 差分运放(Gain=3.3) → Vout = 1.65V + 0.5V×3.3 = 3.3V

ADC层：
3.3V → 12位ADC → 数字值4095

计算层：
ADC值4095 → 电压3.3V
Ibus = (3.3 - 1.65) / (0.1 × 3.3) = 5A ✓
```

### 11.2 完整电压采样链路

```

Vbus = 24V

分压电路：

ADC层：
2.0V → 12位ADC → 数字值2486

计算层：
ADC值2486 → 电压2.0V
Vbus = 2.0 × (100k + 9.1k) / 9.1k = 24V ✓
```

## 十二、总结

### 核心概念

```
✅ 三相桥通过6个MOSFET控制三相电流
✅ PWM调制产生脉冲电流，需要滤波
✅ 母线电流采样是最简单有效的方法
✅ 采样电阻将电流转换为电压
✅ 运放对微弱信号进行放大和调理
✅ ADC将模拟电压转换为数字值
✅ 软件通过公式计算出实际电流电压
```

### 设计原则

```
1. 采样电阻：小阻值、大功率、高精度
2. 运放：高输入阻抗、低噪声、低温漂
3. ADC：足够分辨率和采样率
4. 滤波：模拟+数字双重滤波
5. 校准：上电零点校准，提高精度
```

### 关键公式速查

```
电流：I = (Vadc - Voffset) / (Rs × Gain)
电压：V = Vadc × (R1 + R2) / R2
功率：P = V × I × η
采样率：fs ≥ 2 × fmax
```

**记住：采样电路设计是BLDC电机控制的基础，直接影响控制精度和可靠性！**
