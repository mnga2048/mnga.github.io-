Claude Code 内置命令参考

在 Claude Code 中输入 / 查看所有可用命令 · <arg> 必需参数 · [arg] 可选参数

通用命令▼
命令用途
/help显示帮助和可用命令
/exit退出 CLI 别名：/quit
/login登录到您的 Anthropic 账户
/logout从您的 Anthropic 账户登出
/doctor诊断并验证 Claude Code 安装和设置
/release-notes查看完整的更改日志
对话管理▼
命令用途
/clear清除对话历史记录并释放上下文 别名：/reset、/new
/compact [instructions]压缩对话，可选择性地提供焦点说明
/context将当前上下文使用情况可视化为彩色网格
/copy [N]将最后一个助手响应复制到剪贴板，N 表示第 N 个最新响应
/export [filename]将当前对话导出为纯文本
/branch [name]在此点创建当前对话的分支 别名：/fork
/resume [session]按 ID 或名称恢复对话 别名：/continue
/rename [name]重命名当前会话
/rewind将对话和/或代码倒回到上一个点 别名：/checkpoint
/btw <question>提出快速附加问题，无需添加到对话中
模型与设置▼
命令用途
/model [model]选择或更改 AI 模型
/effort [low|medium|high|max|auto]设置模型工作量级别
/config打开设置界面 别名：/settings
/theme更改颜色主题（浅色/深色/色盲友好/ANSI）
/fast [on|off]切换快速模式
/vim在 Vim 和普通编辑模式之间切换
文件与项目▼
命令用途
/add-dir <path>向当前会话添加新的工作目录
/diff打开交互式差异查看器，显示未提交的更改
/init使用 CLAUDE.md 指南初始化项目
/security-review分析待处理更改以查找安全漏洞
账户与使用▼
命令用途
/cost显示令牌使用统计信息
/usage显示计划使用限制和速率限制状态
/stats可视化每日使用情况、会话历史记录、连续记录和模型偏好
/upgrade打开升级页面以切换到更高的计划层级
/extra-usage配置额外使用量以在达到速率限制时继续工作
/privacy-settings查看和更新隐私设置（仅 Pro 和 Max 计划）
/passes与朋友分享一周免费的 Claude Code
集成与连接▼
命令用途
/mcp管理 MCP server 连接和 OAuth 身份验证
/ide管理 IDE 集成并显示状态
/chrome配置 Claude in Chrome 设置
/desktop在 Claude Code Desktop 应用中继续会话（仅 macOS/Windows）别名：/app
/mobile显示二维码以下载 Claude 移动应用 别名：/ios、/android
/remote-control使此会话可从 claude.ai 远程控制 别名：/rc
/remote-env为网络会话配置默认远程环境
/install-github-app为存储库设置 Claude GitHub Actions 应用
/install-slack-app安装 Claude Slack 应用
GitHub 相关▼
命令用途
/pr-comments [PR]获取并显示 GitHub PR 的评论（需要 gh CLI）
权限与安全▼
命令用途
/permissions查看或更新权限 别名：/allowed-tools
/sandbox切换 sandbox mode
/hooks查看工具事件的 hook 配置
自定义与扩展▼
命令用途
/agents管理 agent 配置
/skills列出可用的 skills
/plugin管理 Claude Code plugins
/reload-plugins重新加载所有活跃 plugins
/memory编辑 CLAUDE.md 内存文件，管理 auto-memory
/keybindings打开或创建快捷键配置文件
其他▼
命令用途

/insights生成会话分析报告

/status打开设置界面（状态选项卡）
/statusline配置 Claude Code 的状态行
/color [color|default]设置提示栏颜色
/voice切换推送通话语音听写
/tasks列出并管理后台任务
